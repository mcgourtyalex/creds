Engineer's Home Page Google Analytics
Features:
  • Target information: The Engineer's Home Page serves to keep the Autodesk community consistent in practices and collaboration. Understanding the visibility of the information that the EHP provides is tantamount to keeping the engineering community well informed.
  • No content left unseen: Other plugins won't show you a complete set of data - the EHP analytics dashboard plugin provides a total view for the scope that each new piece of content is reaching.
  • Spread valuable tools: Understanding which resources are most valuable to engineers will help keep the community as informed as possible.
Using the plugin:
  • To view the chart of pageviews, graph of data, and map from the past week, month, or year, simply select from the pulldown at the top and browse the list or graph that populate. For a more specific time range, use the slider.
  • To find further metrics for each page, select a page from the list or type the pageâ€™s title verbatim. Then, enter the sample size of number of days before the current date, choose a query type, and click fetch data.
  • To manipulate the graph, select a timezone with the pulldown or slider. To zoom, scroll over the area of the graph you'd like to zoom in on. To zoom out, right click.
Client ID:
IT IS NECESSARY FOR EACH WORDPRESS PAGE HOSTING THE PLUGIN TO HAVE A UNIQUE CLIENT ID ASSOCIATED WITH THE EHP.AUTO ACCOUNT.

Here are the steps to create a new Client ID if necessary:
  • Create a new project at the Google Developer's Console
  • Visit 'APIs' under 'APIs & Auth'
  • Turn on the Analytics API
  • Visit 'Credentials' under 'APIs & Auth'
  • 'Create new Client ID'
  • Choose Web Application and enter your website's URL or localhost in 'Authorized Javascript Origins'
